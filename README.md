# tools-platform-ui

Clean React frontend for Tools Platforms, a SEO-first website of practical online tools.

## Stack

- React
- TypeScript
- Vite
- ESLint
- Plain CSS

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

## Local Development

```bash
npm install
npm run dev
```

Default local URL:

```txt
http://localhost:5173
```
